What is my project?

I was given this project by employers from RTVA (Radio and Television of Amstelveen) in Amsterdam. We luckily get to choose the project that we want to work on. The team and I decided to go for second project from a total of five. For this one we need to build a program that would control our live stream, switching between cameras based on audio-input going through the mics. Project is done but I'll keep running this service and work on optimizing my code. There's no real reason why other than I liked working on this project and want to make it work as good as possible.
